  1. O que é HTML. < Voltar

    2. O HTML é uma linguagem de marcação de HiperTexto utilizada para estruturar um documento ou página para a web.

    Ou seja, ao visualizar uma página na internet, você verá a renderização de um código criado em HTML em conjunto com outras linguagens.

  2. Classes e IDs HTML < Voltar

    3. Em HTML, uma classe é uma maneira de identificar um elemento como pertencente a um determinado grupo de elementos.
    Um nome de classe pode ser usado várias vezes em uma única página da Web e cada elemento pode pertencer a várias classes.
    Para especificar uma classe para um elemento, você usa o < class > atributo na tag de abertura do elemento.

    Um ID é uma maneira de identificar exclusivamente um elemento em uma página da web.
    Um ID só pode ser usado uma vez por página da Web e cada elemento pode ter apenas um ID.
    Para especificar um ID para um elemento, você usa o < id > atributo na tag de abertura do elemento.

    Classes e IDs são úteis para aplicar estilos a elementos específicos em uma página da Web e são parte fundamental das linguagens HTML e CSS.

  3. Tag Font < Voltar

    4. A tag < font > foi usada no HTML 4 para especificar a face da fonte, o tamanho da fonte e a cor do texto.

    Exemplo : < font color="#0000FF"> Este texto tem a cor azul. < /font >

    No exemplo acima, estamos especificando a cor da fonte para azul usando o atributo color.
    Então já não usamos mais a tag < font > no HTML5.

  4. Tags Sub e Sup < Voltar

    5. A tag < sub > é usada para colocar o texto em subscripto (indicado por um índice pequeno abaixo do texto).

    Ex: H2O ( Onde apenas o 2 está dentro da tag )

    A tag < sup > é usada para colocar o texto em sobrescrito (indicado por um índice pequeno acima do texto).

    Ex: 30m2 ( Onde apenas o 2 está dentro da tag )
